2.2.1c — Arrays
Component 2: Programming  ·  ~20 min

What is an Array?

An array is a data structure that stores multiple values of the SAME data type in a single named variable. Each value is accessed by its index (position number).

Zero-indexed: In OCR pseudocode, arrays start at index 0. The first element is array[0], the second is array[1], and so on.

1D Array — declaring and using:

scores = [85, 72, 91, 68, 79] // 5 elements, indices 0-4 print(scores[0]) // prints 85 (first element) print(scores[4]) // prints 79 (last element) scores[2] = 95 // update element at index 2 (was 91, now 95) // Length: len(scores) = 5 // Last valid index: len(scores) - 1 = 4

Iterating Through Arrays

Using a FOR loop to process all elements:

names = ["Alice", "Bob", "Charlie", "Dave"] // Print all names for i = 0 to len(names) - 1 print(names[i]) next i // Calculate total of scores total = 0 for i = 0 to len(scores) - 1 total = total + scores[i] next i average = total / len(scores) print(average)

Finding the maximum value:

nums = [3, 7, 2, 9, 5] maxVal = nums[0] // assume first is largest for i = 1 to len(nums) - 1 if nums[i] > maxVal then maxVal = nums[i] endif next i print(maxVal) // prints 9

2D Arrays

A 2D array is a grid (table) of values — like a spreadsheet. Accessed using two indices: [row][column].

2D Array example:

grid = [[1, 2, 3], [4, 5, 6], [7, 8, 9]] print(grid[0][0]) // 1 (row 0, col 0) print(grid[1][2]) // 6 (row 1, col 2) print(grid[2][1]) // 8 (row 2, col 1) // Iterate through all elements for row = 0 to 2 for col = 0 to 2 print(grid[row][col]) next col next row

Common Mistakes

  • Using index 1 for the first element — arrays are ZERO-indexed in OCR: first = array[0]
  • Accessing array[len(array)] — the last valid index is len(array)-1; array[len] is out of bounds
  • Storing different data types in one array — arrays should store the SAME data type
  • For loop from 1 to len — should be 0 to len(array)-1 for all elements
