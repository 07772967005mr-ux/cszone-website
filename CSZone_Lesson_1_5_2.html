1.5.2 — Utility Software
Component 1: Computer Systems  ·  ~18 min

What is Utility Software?

Utility software is system software designed to help manage, maintain, and optimise a computer system. Unlike application software (which performs user tasks) or the OS (which manages hardware), utilities perform background maintenance tasks.

Types of Utility Software

Encryption software

Converts data into an unreadable format (ciphertext) using a key. Protects data stored on a device — if the device is stolen, encrypted data cannot be read without the key. Examples: BitLocker, VeraCrypt.

Defragmentation software

Over time, files on a hard disk become fragmented — split into pieces stored in different locations. Defragmentation rearranges file fragments so they are stored contiguously, speeding up read times. Note: NOT needed/used for SSDs.

Data compression software

Reduces file size by encoding data more efficiently. Two types:

  • Lossless: original data perfectly recoverable (ZIP, PNG, FLAC)
  • Lossy: some data permanently removed (JPEG, MP3, MP4) — smaller files but some quality lost

Backup software

Creates copies of data to protect against loss from hardware failure, ransomware, or accidental deletion. Types:

  • Full backup: copies everything — slow but complete
  • Incremental backup: only backs up files changed since the last backup — fast
  • Differential backup: backs up files changed since the last full backup

Anti-malware / antivirus

Scans files, processes, and emails for malicious software. Quarantines or deletes threats. Must be kept up to date with latest malware signatures.

Firewall software

Monitors and filters network traffic based on rules. Can be software (installed on device) or hardware (dedicated router/appliance). Blocks unauthorised access.

Common Mistakes

  • Saying defragmentation helps SSDs — it doesn't; SSDs don't fragment like HDDs and defragging can damage them
  • Confusing compression types — lossy permanently removes data; lossless is fully reversible
  • Saying "backup" without explaining what it is or where data is stored
  • Confusing anti-malware and firewalls — anti-malware scans files; firewalls filter network traffic
