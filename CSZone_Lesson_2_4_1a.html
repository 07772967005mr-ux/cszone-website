What is Boolean Logic?

Boolean logic deals with values that are either TRUE (1) or FALSE (0). Logic gates are electronic components that implement Boolean operations — they take one or more binary inputs and produce a single binary output.

OCR requires: AND, OR, NOT gates — symbols, function, and how to combine them. XOR is NOT required at GCSE but may appear.

Boolean Expressions

Logic gate circuits can be written as Boolean expressions. OCR uses the following notation:

  • AND: A · B   (dot, or sometimes A AND B)
  • OR: A + B   (plus sign, or A OR B)
  • NOT: Ā   (bar over the variable, or NOT A)

Example circuit expression:

A circuit with: A and B go into an AND gate, whose output goes into a NOT gate:

Output = NOT(A AND B) = (A · B) with bar over it

This is a NAND gate — output is 0 ONLY when both A=1 and B=1.

Common Mistakes

  • Confusing AND with OR — AND requires ALL inputs to be 1; OR only requires ONE
  • Forgetting NOT has only ONE input — it cannot have two inputs
  • Saying AND output is 1 when "at least one" input is 1 — that describes OR
  • Drawing the wrong gate symbol — AND is flat-left D-shape; OR has a curved left edge
  • Confusing 1=TRUE and 0=FALSE with actual numbers
