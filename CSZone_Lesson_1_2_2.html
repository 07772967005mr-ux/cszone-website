1.2.2 — Secondary Storage
Component 1: Computer Systems  ·  ~20 min

What is Secondary Storage?

Secondary storage is non-volatile, permanent storage used to hold data and programs when they are not being actively used by the CPU. Unlike primary storage (RAM), secondary storage retains data when the power is off.

It is needed because RAM is volatile and loses all data when the computer is switched off. Secondary storage is also far cheaper per GB than primary storage.

Key distinction: Primary storage = fast, temporary, directly accessed by CPU. Secondary storage = slower, permanent, must be loaded into RAM before the CPU can use it.

Types of Secondary Storage

1. Magnetic Storage — HDD (Hard Disk Drive)

Hard Disk Drive (HDD)

  • Uses spinning magnetic platters to store data
  • Read/write head moves across the platter to access data
  • Very large capacity (1TB–20TB+) at a low cost per GB
  • Slower than SSDs due to mechanical moving parts
  • Vulnerable to physical shock/damage (moving parts)
  • Produces some noise and heat
  • Used for: desktop computers, servers, backup storage

2. Solid-State Storage — SSD (Solid State Drive)

Solid State Drive (SSD)

  • Uses flash memory (no moving parts)
  • Much faster than HDDs — shorter boot and load times
  • Silent, lighter, more robust (no moving parts to damage)
  • More expensive per GB than HDDs
  • Limited number of write cycles (but typically lasts many years)
  • Used for: laptops, ultrabooks, smartphones, tablets, cameras

3. Optical Storage

Optical Discs (CD, DVD, Blu-ray)

  • Data stored as pits and lands on a disc, read by laser
  • CD: ~700MB, DVD: 4.7GB–8.5GB, Blu-ray: 25GB–50GB
  • Portable, cheap to produce in bulk
  • Easily scratched/damaged; can degrade over time
  • Read speeds much slower than HDDs or SSDs
  • Used for: distributing software/films/games, archiving data

4. Magnetic Tape

Magnetic Tape

  • Data stored magnetically on a long tape ribbon
  • Sequential access only — must read through tape to reach data
  • Very large capacity at extremely low cost per GB
  • Very slow for random access; fast for sequential backup/restore
  • Used for: large-scale data archiving, disaster recovery backups
  • Still widely used in enterprise/cloud data centres

Choosing the Right Storage

Different situations call for different storage types. Consider: capacity needed, speed required, cost, portability, and environment.

  • Need speed? → SSD
  • Need large, cheap storage? → HDD
  • Distributing software/media to many people? → Optical
  • Long-term archiving of huge datasets? → Magnetic tape

Common Mistakes

  • Saying secondary storage is volatile — it is non-volatile
  • Confusing SSD with RAM — both are fast but SSD is non-volatile secondary storage
  • Saying optical discs have moving parts — the disc spins but the data is read by a laser, not a physical head touching the disc
  • Forgetting magnetic tape uses sequential access (not random access like HDD/SSD)
