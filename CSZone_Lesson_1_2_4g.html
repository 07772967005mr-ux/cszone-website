1.2.4g — Representing Sound
Component 1: Computer Systems  ·  ~20 min

How Sound is Stored Digitally

Sound is an analogue signal — a continuous wave. To store it digitally, it must be sampled: the amplitude (loudness) of the sound wave is measured at regular intervals. Each measurement is converted to a binary number and stored.

Sampling = measuring the amplitude of a sound wave at regular time intervals and converting each reading to binary.

Sample Rate

The sample rate (or sampling rate) is how many samples are taken per second. Measured in Hertz (Hz) or kilohertz (kHz).

Examples:

  • CD audio: 44,100 Hz (44.1 kHz) — 44,100 samples per second
  • Telephone: 8,000 Hz (8 kHz)
  • Professional audio: 96,000 Hz (96 kHz)

Effect of sample rate:

Higher sample rate = more samples per second = more accurate representation of the original wave = better sound quality = larger file size.

Lower sample rate = fewer samples = less accurate wave = worse quality = smaller file size.

Bit Depth (Sample Resolution)

Bit depth (also called sample resolution) is the number of bits used to store each sample. More bits = more possible amplitude values = smoother, more accurate sound.

8 bits2⁸ = 256Low quality audio

Calculating Audio File Size

Formula: File size (bits) = Sample rate × Bit depth × Duration (seconds) × Number of channels
Divide by 8 for bytes.

Example 1: Mono, 44,100 Hz, 16 bits, 30 seconds

= 44,100 × 16 × 30 × 1
= 21,168,000 bits
÷ 8 = 2,646,000 bytes
÷ 1024 ÷ 1024 ≈ 2.52 MB

Example 2: Stereo, 44,100 Hz, 16 bits, 60 seconds

= 44,100 × 16 × 60 × 2
= 84,672,000 bits
÷ 8 = 10,584,000 bytes
÷ 1024 ÷ 1024 ≈ 10.09 MB

Channels: Mono vs Stereo

  • Mono (1 channel) — one audio track. Multiply by 1.
  • Stereo (2 channels) — separate left and right tracks. Multiply by 2 — doubles the file size.

Common Mistakes

  • Confusing sample rate (Hz) with bit depth (bits per sample)
  • Forgetting to multiply by the number of channels for stereo
  • Forgetting to divide by 8 when converting bits to bytes
  • Using kHz without converting to Hz (44.1 kHz = 44,100 Hz) in calculations
  • Saying higher sample rate = more bits per sample (it means more samples per second)
