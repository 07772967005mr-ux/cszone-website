Architecture of the CPU
📖 Lesson notes ▶ Video ⊞ Slides 📝 Worksheet ✎ Quiz 🔑 Key terms
📖 Key concepts
The Fetch-Execute Cycle
The CPU continuously repeats the fetch-execute cycle to process instructions
Fetch → the PC holds the address of the next instruction; it is copied to the MAR
Decode → the CU interprets the instruction held in the CIR
Execute → the ALU carries out arithmetic or logical operations
