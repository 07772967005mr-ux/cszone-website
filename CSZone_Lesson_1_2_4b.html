1.2.4b — Hexadecimal
Component 1: Computer Systems  ·  ~25 min

What is Hexadecimal?

Hexadecimal (hex) is a base 16 number system. It uses 16 symbols: the digits 0–9 and the letters A–F (where A=10, B=11, C=12, D=13, E=14, F=15).

Key insight: Each hex digit represents exactly 4 binary digits (a nibble). This makes converting between hex and binary very easy.

Why Use Hexadecimal?

Hex is used because it provides a compact, human-readable representation of binary data.

Uses of Hexadecimal

  • Memory addresses — RAM and CPU registers use hex addresses (e.g. 0x1F4A)
  • Colour codes — HTML/CSS colours like #FF5733 (R=FF, G=57, B=33)
  • Error codes — Windows error codes (e.g. 0x0000007F)
  • MAC addresses — unique hardware identifiers (e.g. A1:B2:C3:D4:E5:F6)
  • IPv6 addresses — internet addresses using hex groups
  • Assembly/machine code — representing opcodes and data

Two hex digits can represent a full byte (8 bits): e.g. FF₁₆ = 11111111₂ = 255₁₀

Converting Hex to Denary

Hex place values are powers of 16: 16⁰=1, 16¹=16, 16²=256, 16³=4096

Worked Example: Convert 3F₁₆ to denary

3F has two digits: leftmost = 3, rightmost = F (=15)
= (3 × 16) + (15 × 1)
= 48 + 15
= 63

Worked Example: Convert 2A₁₆ to denary

2A: 2 in 16s column, A (=10) in 1s column
= (2 × 16) + (10 × 1)
= 32 + 10
= 42

Worked Example: Convert 1B4₁₆ to denary

Place values: 256, 16, 1
1=1, B=11, 4=4
= (1 × 256) + (11 × 16) + (4 × 1)
= 256 + 176 + 4
= 436

Converting Denary to Hex

Repeatedly divide by 16 and note the remainders (in reverse order).

Worked Example: Convert 200 to hex

200 ÷ 16 = 12 remainder 8 → remainder = 8
12 ÷ 16 = 0 remainder 12 → remainder = C
Read remainders bottom to top: C8₁₆

Check: (12 × 16) + 8 = 192 + 8 = 200 ✓

Converting Hex ↔ Binary (via Nibbles)

The easiest method: each hex digit = exactly 4 binary bits (a nibble).

Hex to Binary: Convert A7₁₆ to binary

A = 10 = 1010
7 = 7  = 0111
A7₁₆ = 10100111₂

Binary to Hex: Convert 11001101₂ to hex

Split into nibbles: 1100 | 1101
1100 = 12 = C
1101 = 13 = D
= CD₁₆
Shortcut: To convert an 8-bit binary number to hex, split it into two nibbles and convert each to a hex digit. To go the other way, convert each hex digit to a 4-bit nibble.

Common Mistakes

  • Forgetting that A=10, B=11, C=12, D=13, E=14, F=15
  • Not padding nibbles to 4 bits (e.g. 7 = 0111 not 111)
  • Using wrong place values (hex uses 16s, not 10s)
  • Mixing up hex letters — E is 14, not 15 (F is 15)
