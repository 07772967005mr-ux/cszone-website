2.1.2b — Pseudocode
Component 2: Programming  ·  ~20 min

What is Pseudocode?

Pseudocode is a way of writing an algorithm in structured English that resembles programming but is not tied to any specific programming language. It is used to plan programs and communicate algorithms clearly.

OCR J277 uses its own pseudocode style. You must be able to READ and WRITE OCR pseudocode in the exam.

OCR Pseudocode Reference

Variables and assignment:

x = 5 name = "Alice" total = price * quantity

Input and Output:

name = input("Enter your name: ") print("Hello " + name)

Selection (IF statements):

if score >= 50 then print("Pass") else print("Fail") endif

Iteration — count-controlled (FOR loop):

for i = 1 to 10 print(i) next i

Iteration — condition-controlled (WHILE loop):

while password != "secret" password = input("Enter password: ") endwhile

DO UNTIL loop:

do number = input("Enter a positive number: ") until number > 0

Arrays:

scores = [45, 67, 89, 23, 55] print(scores[0]) // First element scores[2] = 90 // Update element

Subroutines (Procedures and Functions):

procedure greet(name) print("Hello " + name) endprocedure function square(n) return n * n endfunction

Common Mistakes

  • Forgetting endif, endwhile, next i — OCR requires closing keywords
  • Using = for comparison instead of == — in OCR pseudocode use == for comparison
  • Writing Python/real code in pseudocode questions — pseudocode is language-independent
  • Forgetting that array indexes start at 0
